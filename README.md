# Predictive Modelling of Loan Approval ‚Äì Lasiandra Finance Inc.

## üìå Project Overview
This project was developed as part of the **Data Analytical Programming (CT050-3-M-DAP)** module at **Asia Pacific University (APU)**.  
It focuses on building a **predictive loan approval model** for **Lasiandra Finance Inc. (LFI)**, a private financial company in New York, USA.  

The goal is to automate the loan-eligibility process using **data science techniques** and **SAS programming**, reducing human error, improving efficiency, and enhancing decision-making accuracy.

---

## üéØ Objectives
- Analyze customer-related factors influencing loan approval:
  - Gender, marital status, family size, qualification, employment type
  - Candidate income, guarantor income, loan amount, loan history, loan location
- Preprocess and clean datasets to handle missing values.
- Build predictive models for loan approval using **SAS Studio**.
- Generate professional reports and visualizations for business decision-making.
- Demonstrate how **data analytics** can minimize risk and improve profitability in financial institutions.

## üìÇ Dataset
- **Source:** SME loan application records (training dataset).
- **Size:** 614 observations, 13 variables.
- **Variables:**
  - **Categorical:** Employment, Family Members, Gender, Loan Approval Status, Loan Location, Marital Status, Qualification, SME Loan ID.
  - **Numerical:** Candidate Income, Guarantor Income, Loan Amount, Loan Duration, Loan History.

---

## üìä Key Analyses
- **Univariate Analysis:** Distribution of categorical and continuous variables (e.g., marital status, employment, loan amount, income).
- **Bivariate Analysis:** Relationships between variables (e.g., gender vs loan location).
- **Missing Value Imputation:** Handling incomplete records for accurate modelling.
- **Predictive Modelling:** Loan approval prediction using SAS procedures.
- **Visualization:** Loan application workflow, data-science workflow in banking, and conceptual frameworks for risk management.
